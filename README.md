## Installation

#### downloading Project 

```
  git clone https://github.com/MmdBay/pintrest_downloader.git
```
#### Installation in production mode

```
  cd .....
  npm i
  tsc src/index
  node src/index.js
```

## Run Locally

Clone the project

```
  git clone https://github.com/MmdBay/pintrest_downloader.git
```

Go to the project directory

```
  cd project
```

Install dependencies

```
  npm install
```

Start the server

```
  cd .....
  nodemon --exec ts-node src/index.ts
```
